# Vulnerability Test Files

These files contain intentional vulnerabilities for testing Subrunner.

- **sqlInjection.js** - SQL injection
- **xssReflected.js** - XSS
- **hardcodedSecrets.js** - Hardcoded secrets
- **commandInjection.js** - Command injection
- **unsafeEval.js** - Unsafe eval
- **authBypass.js** - Auth bypass patterns

**Do not use in production.**
